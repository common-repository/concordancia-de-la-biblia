== Description ==Utilice el cuadro de búsqueda para insertar cualquier palabra, puede buscar todos las versiones de la biblia disponibles, o elegir una individual.== Installation ==Puedes colocar la concordancia de la biblia sin necesidad de instalar este plugin, con solo copiar este codigo en tu widget como Texto arbitrario o HTML:`<script type="text/javascript" language="javascript" src="https://www.bibliatodo.com/assets/js/wordpress/es/widget-concordancia.js"></script>`o puedes instalar el el plugin y seguir estas indicaciones
1. Sube el archivo a wp-content/plugin2. Activar plugin3. Ir a Apariencia > widgets4. Arrastra el widget al sidebar que quieras.5. Comprueba que se ve correctamente!
   - Widget: Se puede añadir un widget a la barra lateral de la Concordancia de la Biblia, vaya a apariencia - widget "Concordancia de la Biblia"
   - Template: en cualquier lugar de su plantilla inserte este codigo `<script type="text/javascript" language="javascript" src="https://cdn.bibliatodo.com/assets/js/wordpress/es/widget-concordancia.js"></script>`.
